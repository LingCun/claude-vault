| 메서드 | 시점 | 동작 |
|---|---|---|

### 🆕 신규 #2 — `culture.xml` (mybatis XML)

**위치**: `bld-v3/src/main/resources/mapper/culture.xml`

**6개 statement**:

| statement | 타입 | 대상 |
|---|---|---|
| `saveTbtrCulture` | INSERT | TBTR_CULTURE (30+ 컬럼, WORKER='BLD', REG_DNT=NOW yyyyMMddHHmmss) |
| `updateTbtrCultureForCancel` | UPDATE | TBTR_CULTURE.TRX_ST_CD/APP_NO/CTRL_CD (TID 기준) |
| `saveTbusCulture` | INSERT | TBUS_CULTURE (실패 정보: CULTURE_RSLT_CD/MSG + 송신값 사본) |
| `selectToCancelCultureInfo` | SELECT | TBTR_CULTURE A + TBTR_MSTR B + TBTR_HIST C 조인 (card.xml 의 `selectToCancelCardInfo` 패턴) |
| `selectCultureTransByAppNo` | SELECT | TBTR_CULTURE A + TBTR_MSTR B 조인, APP_NO/TR_DT/MID/GOODS_AMT 매칭 |
| `selectCultureTransByEdiNo` | SELECT | TBTR_CULTURE A + TBTR_MSTR B LEFT JOIN, VAN_CD/EDI_NO 매칭 |

**컬럼 매핑 출처**: [[2026-05-18_컬처캐쉬_연동_분석보고서_v01.0]] §4.3 (TBTR_CULTURE) / §4.4 (TBUS_CULTURE).
