## 3. 채택한 아키텍처 — GHA + Claude Code Action

> 5/18 가이드 초안에는 옵션 A (로컬 Claude CLI + Task Scheduler) 추천이었으나, 실제 구현은 **옵션 B 진화형**으로 확정 — GHA 에서 `anthropics/claude-code-action@v1` 가 Pro/Max OAuth 토큰으로 작동 → API 비용 0.

```
GitHub Actions cron (21:00 UTC US / 07:00 UTC KR)
    ↓
1. node scripts/fetch-market.mjs
    → src/data/market-snapshot.json 갱신 (yahoo-finance2 + KRX/DART)
2. Load generation prompt step
    → scripts/generation-prompt.md → step output PROMPT
3. anthropics/claude-code-action@v1
    → Claude Sonnet 4.6 이 prompt 따라:
       ├─ STYLE.md / CLAUDE.md / market-snapshot.json 읽기
       ├─ 최근 같은 카테고리 글 2편 톤 참조
       ├─ WebFetch 로 뉴스 헤드라인 보강
       ├─ 4편 .md 작성 (frontmatter zod 통과)
       ├─ 4개 SVG 썸네일
       ├─ Safety gate (금지어 / 사인오프 / 분량 / 출처 숫자)
       └─ npx astro build 검증
4. claude[bot] 가 main 으로 직접 commit + push
    → Vercel 자동 빌드·배포
```

**Step별 소요** (5/20 US 검증):
- Setup + fetch + install: ~30초
- Claude Code Action 실행: ~18분 (4편 + 썸네일 + 검증)
- 총: 19분 내외

**비용**: Claude Pro/Max 구독 quota 차감 (별도 결제 없음). 일 8편 ≒ quota 적정 범위.

**필수 secrets** (GitHub repo Settings → Secrets):
- `CLAUDE_CODE_OAUTH_TOKEN` — [발급 가이드](https://docs.claude.com/en/docs/claude-code/github-actions)
- `KRX_API_KEY` (선택, 한국 분석 깊이 ↑)
- `DART_API_KEY` (선택, 공시 인용)

**필수 권한** (워크플로 `permissions:` 블록):
- `contents: write`, `pull-requests: write`, `issues: write`
- `id-token: write` — Action OIDC 인증 (없으면 401)

## 4. 안전 게이트 — 자동 발행 시 보호 장치

> [!important] 자동 발행이지만 품질 가드 4중
> 검토 큐 거치지 않고 main 푸시되지만, 본문이 기준 미달이면 `draft: true` 자동 부착 → Astro 빌드에서 미발행 처리.

Prompt 8번 항목의 safety gate (Claude 가 commit 직전 실행):

| 가드 | 조건 | 미달 시 |
|---|---|---|
| 금지어 | "매수 추천", "사세요", "확실히", "반드시 상승", "무조건 상승" | `draft: true` |
| 사인오프 | `— Lincoln` (em-dash + space) 누락 | `draft: true` |
| 분량 | 본문 < 600 단어 | `draft: true` |
| 수치 출처 | 본문 숫자 ↔ market-snapshot.json 불일치 | `draft: true` |
| TODO 마커 | `[TODO: Lincoln 검토]` 잔존 | `draft: true` |

추가 안전망:
- 같은 슬러그 이미 존재 → skip (덮어쓰기 금지, 수동 편집 보호)
- 4편 전부 draft 면 git diff 비어있어 commit 자체 없음 — 워크플로 success 지만 발행 0편

## 7. 비상 매뉴얼

### 수동 dispatch (스케줄 외 즉시 실행)

```powershell
cd C:/claude/lincoln-brief

# 미국 즉시 실행
gh workflow run "Daily Market Brief (US, Claude Code)" --ref main

# 한국 즉시 실행
gh workflow run "Daily Market Brief (KR, Claude Code)" --ref main

# 진행 보기
gh run watch <RUN_ID> --exit-status
```

### 일시 중지

```powershell
# GitHub web: Actions 탭 → 해당 워크플로 → "⋯" → Disable workflow
# 또는 cron 줄을 잠깐 주석 처리하고 push
```

### 잘못 발행된 글 롤백

```powershell
cd C:/claude/lincoln-brief
git revert <commit-hash> --no-edit
git push
```

Vercel 이 revert 커밋 받아 자동 재배포 (~30초).

### 특정 글만 draft 처리 (revert 없이 노출 차단)

해당 `.md` 파일 frontmatter 에 `draft: true` 추가 → commit + push.
