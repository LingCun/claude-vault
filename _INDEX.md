# 볼트 안내

## 폴더 구조

- `개인/` — 일상, 학습, 아이디어, 개인 메모 등
- `업무/` — 회사 관련 모든 내용 (회의, 프로젝트, 업무 메모 등)

## 자동 분류 규칙

1. **기본 2단계 구조**: 개인 / 업무로 먼저 분류
2. **세부 폴더 자동 생성**: 특정 프로젝트나 주제가 반복되면 하위 폴더를 만들어서 정리 (예: `업무/프로젝트A/`, `업무/회의/`)
3. **애매할 때만 확인**: 명확하면 바로 저장, 개인/업무 경계가 흐릿할 때만 한 번 물어봄
4. **회사 맥락**: 대화 중 자연스럽게 파악해 누적 (회사명, 주요 프로젝트, 동료 이름 등)

## 파일 명명 규칙

- `YYYY-MM-DD_주제.md` 형식
- 핵심만 간결하게 작성
