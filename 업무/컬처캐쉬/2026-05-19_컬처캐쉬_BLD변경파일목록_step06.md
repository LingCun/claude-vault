## 2. 파일별 상세

### 🆕 #1 — `CultureConstants.java`

상수 클래스. PM_CD/SPM_CD/PTN_CD + Mock 식별자.

```java
PM_CD             = "32"
SPM_CD            = "07"

MOCK_MEMBER_CODE  = "MOCKCULTURE1"

TRX_ST_NORMAL/CANCEL_PRE/CANCEL_POST = "0"/"1"/"2"

```

### 🆕 #2 — `CommonCultureMethod.java`

`@Component` Mock 응답 합성 유틸.

| 메서드 | 합성 항목 |
|---|---|
| `synthesizeMockApprovalResponse(wTxData)` | appNo (12자리 숫자), ctrlCd (12자리), mbrCtrlCd (TID 가공 25자), levyDt/Tm, levyAmt, levyUnit='건', payType='00', mcashAmt (1,000,000 − goodsAmt), ptnCd='CULT', trxStCd='0', authTypeCd='07', trDt, appAmt, resultCd='0000' |
| `synthesizeMockCancelResponse(wTxData, netCancel)` | cancelCtrlCd (12자리), cancelAmt, cancelDt/Tm, trxStCd='1' (전취소), resultCd='0000' |

### 🆕 #3 — `PayCultureBean.java`

`@Service` 승인 처리 빈. `BizPattern` + `ProcInterface` + `PayInterface`.

| 메서드 | 동작 |
|---|---|
| `beforProcess` | `commonValidateNSetBaseInfoWithQuery` 호출, ptnCd/authTypeCd 세팅 |
| `doProcess` | `commonDoPayProcess(wTxData)` (BizPattern 공통 흐름) |
| `processSendAppl` | **STEP1 Mock**: 8110 송신 skip → `CommonCultureMethod.synthesizeMockApprovalResponse()` |
| `afterPaySuccessAsBiz` | `cultureMapper.saveTbtrCulture(procDataMap)` — TBTR_CULTURE INSERT |
| `afterPayFailAsBiz` | `cultureMapper.saveTbusCulture(procDataMap)` — TBUS_CULTURE INSERT |
| `netCancelAsBiz` | **STEP1 Mock**: 8710 송신 skip → `saveTbusCulture` 로그만 |
| `afterProcess` | resultData 에 appNo, resultCd 세팅 |

### 🆕 #4 — `CancelCultureBean.java`

`@Service` 취소 처리 빈. `BizPattern` + `ProcInterface` + `CancelInterface` + `NoTrxCancelInterface`.

| 메서드 | 동작 |
|---|---|
| `beforProcess` | 기준정보 + 취소정보 검증 |
| `doProcess` | `commonDoCancelProcess(wTxData)` |
| `processSendCancel` | **STEP1 Mock**: 8210 송신 skip → `synthesizeMockCancelResponse()` |
| `afterCancelSuccessAsBiz` | trxStCd='1' 세팅 후 `cultureMapper.updateTbtrCultureForCancel()` |
| `afterCancelFailAsBiz` | `cultureMapper.saveTbusCulture(procDataMap)` |
| `afterProcess` | resultData 에 resultCd 세팅 |

> 컬처캐쉬는 **전체취소 + 망상취소만** 지원 — 부분취소 진입 시 별도 거절 처리는 본 빈 외부 (PaymentOrderReq 등) 에서 사전 차단.
